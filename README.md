# ChukoUriage

中古車の売上を管理するためのJavaアプリケーションです。

## 概要
本アプリケーションは、中古車販売における売上データの集計、売り上げ台数、支店ごと車種ごとの合計金額、車種毎の合計金額を表示するためのシステムです。

## 主な機能
* **登録:** 1台販売毎に支店、車種を登録する機能
* **全データ表示:** 全売上データを表示する機能
* **売上台数:** 支店毎の売り上げ台数を表示する機能
* **車種マスタ:**　車種と価格を新規登録、修正、削除する機能

## 動作確認環境

本アプリケーションは、以下の開発環境および Android Studio のエミュレーター（仮想デバイス）を使用して動作確認を行っています。現時点では、特定のデバイス・OSバージョンでのみ検証済みです。

### 開発ツール
* **IDE:** Android Studio Panda 4 | 2025.3.4
* **Build Tool:** Gradle

### 動作確認済みエミュレーター（Virtual Device）仕様
アプリの挙動は、以下の構成を持つ Android Virtual Device (AVD) 上でのみ動作確認を行っています。

| 項目 | 設定内容 / バージョン | 備考 |
| :--- | :--- | :--- |
| **対象デバイス** | **Pixel 4** | 画面サイズ・解像度の基準 |
| **OS バージョン** | **Android 10.0 (API 29)** | **※現状、本バージョンのみでテストを実施しています** |
| **アーキテクチャ** | arm64-v8a (または x86_64) | お使いのPCのCPUに合わせて選択してください |
| **Database** | H2 Database (インメモリ) | 開発用の組み込みDBを使用（起動時に自動生成） |
---

## エミュレーターでの実行手順

1. **Android Studio** を起動し、本プロジェクトを開きます。
2. 画面右上の **Device Manager**（デバイスマネージャー）を開き、`Pixel 4 - API 29` の仮想デバイス（AVD）を作成・起動します。
3. ターゲットデバイスに起動したエミュレーターが選択されていることを確認し、**Runボタン（緑色の再生アイコン）** をクリックしてアプリケーションを実行します。
